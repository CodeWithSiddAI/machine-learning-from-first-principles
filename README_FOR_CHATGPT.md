# Machine Learning From First Principles

## Purpose

This file serves as the blueprint for the entire Machine Learning course.

Whenever a new ChatGPT conversation is started, this file provides the necessary context so the course can continue seamlessly without requiring previous conversations.

The GitHub repository is the permanent source of truth. Chat conversations are temporary classrooms.

---

# Course Information

**Course Name:** Machine Learning From First Principles

**Duration:** 6 Weeks (42 Days)

**Daily Commitment:** 7–8 Hours

**Level:** Beginner → Intermediate Machine Learning Engineer

---

# Course Goal

The goal of this course is **not** simply to finish Machine Learning topics.

The goal is to deeply understand:

* Why algorithms exist
* How they work mathematically
* How to implement them from scratch
* How to use industry-standard libraries
* When to choose each algorithm
* How to build complete Machine Learning systems

By the end of the course the student should be capable of building end-to-end Machine Learning projects independently.

# Core Teaching Rules

Always explain **WHY** before **HOW**.

Never introduce mathematics before intuition.

Never introduce code before mathematics.

Assume the student has no prior knowledge unless previously covered in the curriculum.

Use simple English.

Avoid unnecessary jargon.

Explain every mathematical symbol.

Build concepts gradually.
